# Codex Prompt: Tags Redesign — Move to Top, Make Clickable

## Repo Context
Repo: ~/anky-seed
Backend: backend/server.ts (Bun/TypeScript on Railway)
iOS: apps/ios/Anky/Features/Reveal/

## Current Behavior
Tags are generated by the LLM as inline markdown text within the reflection body. The prompt at server.ts:2078 says: "If you include tags, write them as markdown text." They render as regular text in SelectableReflectionText (RevealView.swift:906-1107). No structured data, no navigation.

## Desired Behavior
1. Tags appear as styled pills at the TOP of the reflection, above the content
2. Tags are clickable — tapping navigates to a list of all sessions with that tag
3. Tags are stored as structured data alongside the reflection

## Tasks

### Backend (server.ts)

1. **Modify buildStorytellerPrompt() (line 2051-2086):**
   - Instead of "write tags as markdown text", instruct the LLM to output tags as a structured JSON block at the very top of the response:
   ```
   Before the markdown document, output a JSON block:
   {"tags": ["tag1", "tag2", "tag3"]}
   Then output the markdown document starting with # H1 title
   ```
   - Limit to 3-5 tags max
   - Tags should be theme/emotion words (not kingdom names for now, keep it simple)

2. **Modify parseMirrorResponse() (line 2039-2048):**
   - Parse the JSON tags block from the raw response before the markdown
   - Return tags alongside reflection: `{ title, reflection, tags: string[] }`

3. **Modify the reflection SSE event (line 2209-2212):**
   - Include tags in the `reflection` event payload

### iOS — Data Model

4. **Update LocalReflection (ReflectionStore.swift:3-11):**
   - Add `tags: [String]` field
   - Update JSON encoding/decoding

5. **Update MirrorResponsePayload (MirrorClient.swift:76-81):**
   - Add `tags: [String]?` field

### iOS — UI

6. **Update ReflectionScrollPage (RevealView.swift:581-613):**
   - At the top of the scroll view, before the reflection text, render tags as horizontal scrollable pill buttons
   - Styling: rounded pills, subtle background, gold text, matching Anky palette
   - Each pill is a NavigationLink to TagSessionsListView

7. **Create TagSessionsListView (new file):**
   - Fetch all sessions from SessionIndexStore that match the tag
   - Display as a list: date, session title preview, word count
   - Tapping a session navigates to its RevealView
   - Include a "Back" button

8. **Update SessionIndexStore (SessionIndexStore.swift):**
   - Add tag indexing: when saving a reflection, index its tags
   - Add `sessionsWithTag(_ tag: String) -> [SavedAnky]` method

### iOS — ViewModel

9. **Update RevealViewModel.askAnky() (line 174-239):**
   - Extract tags from MirrorResponsePayload and save with LocalReflection
